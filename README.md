## 📖 Project Overview

This project demonstrates an end-to-end Supply Chain & Inventory Analytics solution for a retail business using Python, PostgreSQL, SQL, and Tableau.

The objective is to transform raw operational data into actionable business insights by performing data cleaning, feature engineering, exploratory data analysis (EDA), SQL-based business analysis, and interactive dashboard development.

The project simulates a real-world analytics workflow followed by data analysts and business intelligence professionals.

---

## 🎯 Business Problem

Retail organizations often struggle with:

- Overstocked inventory leading to high holding costs
- Stockout situations causing lost sales
- Inefficient warehouse utilization
- Inconsistent supplier performance
- Inaccurate demand forecasting
- Lack of centralized business reporting

This project addresses these challenges by building analytical datasets and business dashboards to support data-driven decision-making.

---

## 🎯 Project Objectives

- Clean and validate multiple supply chain datasets
- Engineer business-focused inventory metrics
- Perform comprehensive exploratory data analysis
- Identify inventory, warehouse, supplier, and forecasting risks
- Write business-oriented SQL queries using PostgreSQL
- Prepare analytical datasets for Tableau dashboards
- Build an executive-level Supply Chain Analytics Dashboard

# 📈 Key Business Insights

Some key findings from the analysis include:

- Most inventory remains in a healthy stock position.
- Overstock inventory contributes significantly to inventory holding value.
- Warehouse efficiency varies across facilities despite similar utilization levels.
- High-risk suppliers exhibit lower delivery performance and higher defect rates.
- Demand forecasting accuracy averages approximately 87%, with a subset of products requiring forecasting improvements.
- Supply Chain Health Score provides a consolidated performance indicator for executive monitoring.

---

# 💡 Business Recommendations

Based on the analysis:

- Reduce excess inventory for slow-moving products.
- Prioritize replenishment for stockout-risk SKUs.
- Improve forecasting models for high-risk products.
- Monitor supplier performance continuously.
- Optimize warehouse operations using efficiency metrics.
- Use executive dashboards for real-time decision support.
